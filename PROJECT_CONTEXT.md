# Smithware Android MVP Template Context

## App

- Name: Smithware Android MVP Template
- Studio: Smithware Studios
- Package: `com.smithware.mvpstarter`
- Repo: `https://github.com/BadBagger/smithware-android-mvp-template`
- Current release target: template repo, not a DevHub app release

## Purpose

This repo is a buildable starter for local-first Smithware Android apps. It
packages the common Compose shell, Room persistence, DataStore preferences,
standard screens, demo data, validation, edit/delete/archive actions, prompt
export, light/dark mode, and release handoff docs.

When using it for a new app, replace all template identifiers and publish the
new app repo with APK assets. Do not connect this template itself as a DevHub
installable app.

## Required App Customization

- App name and package
- Product-specific data model names and seed data
- Dashboard and export copy
- App icon and DevHub listing assets
- `AGENTS.md`, `PROJECT_CONTEXT.md`, README, release tag, and APK names
