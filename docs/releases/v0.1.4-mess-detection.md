# TidyPilot v0.1.4 Mess Detection

## Highlights

- Improves the local Room Photo Scan flow with mess-level detection, confidence,
  detected zones, review copy, and practical task suggestions.
- Keeps photo analysis local-first with no cloud upload, no account, and no paid
  API dependency.
- Adds Room database migration support for richer scan and scan issue metadata.
- Adds scanner tests for mess-level mapping, low-energy task filtering, room
  score effects, and online-reference fixture scenarios.
- Adds offline vision dataset tooling for future on-device computer vision:
  manifests, Kaggle import, Open Images sampling, pixel feature extraction, and
  local review contact sheets.
