# TidyPilot v0.1.5 Dashboard Routines

## Highlights

- Refreshes the app color profile to match the TidyPilot logo: aqua, mint,
  cream, yellow, coral, and warmer dark surfaces.
- Adds a dashboard "Doable for your energy" todo card.
- Adds a dashboard "Routine autopilot" card for repeating chores.
- Explains repeat behavior in Add/Edit Task before saving.
- Shows routine autopilot behavior on Task Detail.
- Replaces several smart punctuation and bullet separators with plain text for
  cleaner Android rendering.
