# TidyPilot v0.1.6 UI Polish

## Highlights

- Refines the Today dashboard into a calmer home reset command center.
- Adds a primary "Start recommended reset" action with scan and check-in as
  secondary actions.
- Makes top-level headers more compact while keeping TidyPilot branding.
- Improves Quick Add with clearer Task/Room/Shift tabs and categorized compact
  templates.
- Reworks Room Management cards with softer status badges, compact room facts,
  and fewer large action buttons.
- Redesigns Reports into visual sections for weekly snapshot, room highlights,
  energy/work impact, follow-up, and local export actions.
